# Calculadora de energía solar fotovoltaica

- [Código fuente](./src/)
- [Pruebas unitárias](./spec/)

## Instalar

```javascript
npm install
```

## Correr las pruebas unitárias

### Una vez

```javascript
npm test
```

### Interactiva

```javascript
npm run test-server
```

Después abrir el navegador en http://localhost:8888
